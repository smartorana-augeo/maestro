# Maestro agents

Specialized agents organized by category. Invoke in Claude Code for multi-turn, focused tasks.

## workflow

- **felix** — Intelligence coordinator, agent orchestrator, and product/project manager. Decomposes complex initiatives, delegates to specialized subagents, synthesizes results, and drives delivery (Opus)
- **project-manager** — Project planning, task breakdown, progress tracking, delivery management
- **debugger** — Bug diagnosis, root cause analysis, error logs, stack traces, and production failures
- **git-workflow-manager** — Git workflows, branching strategies, merge resolution, repository management
- **github-actions-expert** — GitHub Actions for CI/CD and workflow automation
- **documentation-engineer** — Technical and API documentation, docs-as-code

## architecture

- **technical-architect** — System design, architecture decisions, technical planning
- **cloud-architect** — Multi-cloud (AWS, Azure, GCP), scalable architectures, cost and compliance

## quality

- **code-reviewer** — Code review with focus on security, quality, performance, best practices
- **refactoring-specialist** — Safe code transformation, design patterns, maintainability
- **ux-researcher** — User insights, usability testing, accessibility, data-driven design decisions
- **chaos-engineer** — Chaos testing, fault injection, resilience validation, failure mode analysis

## frameworks

- **nextjs-expert** — Next.js, SSR/SSG, serverless, React optimization
- **playwright-expert** — Playwright test automation, reliable test suites
- **graphql-expert** — GraphQL API design, schemas, query optimization
- **typescript-expert** — TypeScript/JavaScript, type system, ES2023+, full-stack, Node, browser, build optimization

## database

- **mongodb-expert** — MongoDB operations, Mongoose ODM, schema design, indexing, aggregations, query optimization
- **postgres-expert** — PostgreSQL, Prisma ORM, complex SQL, indexing, migrations, type-safe queries
- **sql-pro** — Advanced SQL optimization, schema design, performance tuning across PostgreSQL, MySQL, SQL Server, and Oracle

## development

- **ui-ux-designer** — UI/UX design, wireframes, prototypes, and design system implementation
- **js-node-developer** — Node.js/TypeScript backend (GraphQL, REST, MongoDB, PostgreSQL)
- **js-react-developer** — React frontend (Next.js, Apollo GraphQL, MUI)
- **js-reactnative-developer** — React Native mobile (Redux, Apollo GraphQL, Navigation)
- **fullstack-developer** — End-to-end features spanning backend + frontend, GraphQL contract design
- **dotnet-developer** — .NET architecture, implementation, testing, security, and performance
- **coldfusion-developer** — ColdFusion/Lucee CFML debugging, modernization, performance, and security

## devops

- **bash-expert** — Defensive Bash, CI/CD pipelines, production automation
- **grafana-expert** — Grafana dashboards, visualization, alerting
- **loki-expert** — Loki log aggregation and querying

Each agent has a detailed `.md` file in its category folder with role, capabilities, and usage.
